# Local Cover Pool

Place your default non-WeChat cover images in this directory.

- Supported formats: `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`
- Recommended names: `cover_01.png`, `cover_02.png`, `cover_03.png`
- The desktop workbench will detect files here automatically

If you use a custom cover directory, set `assignment.cover_dir` in `config.json`.
